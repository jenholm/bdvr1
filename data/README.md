# Data directory

## Structure

```
data/
├── raw/         Source input files (SPARC quality-1 inputs, xGASS M75 preprocessing inputs)
├── external/    Manually downloaded external catalogs (not committed)
├── processed/   Cleaned intermediate tables (generated by scripts)
└── derived/     Paper-specific derived products (generated by scripts)
```

## Committed input files

This repository includes the following input files for reproducibility:

- `data/raw/sparc/sparc_quality1_rich_inputs.csv` — SPARC quality-1 galaxies (87 galaxies, Q=1). Source: Lelli+2016.
- `data/raw/xgass/m75_xgass_coherence_scores.csv` — historical M75 preprocessing table: xGASS H I-detected galaxies (363) with pre-computed BTFR residuals, `f_atm_total`, and organization proxy inputs. Source: xGASS catalog (Catinella+2018).
- `data/raw/xgass/xgass_representative_sample.csv` — xGASS representative sample catalog metadata.

### Obtaining original catalogs

#### SPARC
Download the full SPARC catalog from http://astroweb.cwru.edu/SPARC/

#### xGASS
Download the xGASS representative sample from https://xgass.astro.umd.edu/

Place additional files in `data/raw/xgass/` and run:
```bash
python scripts/01_prepare_inputs.py
```

## Key data definitions

### BTFR residual definitions

**xGASS**: `btfr_residual_dex` in `xgass_clean.csv` is copied from the M75 input column `btfr_offset_dex`. These residuals reproduce as a global OLS fit of logVflat on logMbar (slope ≈ 0.279, intercept ≈ −0.689), **not** as the McGaugh 2012 calibration (Mbar = 50 × Vflat⁴). The paper's cross-validated xGASS result refits the BTFR inside folds downstream (script 05, `cross_validated_btfr_residuals`).

**SPARC** has two BTFR residual definitions in `btfr_residuals_sparc.csv`:
- `btfr_residual_dex` — OLS fit on SPARC quality-1 sample: logV = −0.511 + 0.260 × logMbar.
- `btfr_mcgaugh2012_residual_dex` — McGaugh 2012 calibration: logV_pred = 0.25 × (logMbar − log₁₀(50)).

### Proxy definitions

**SPARC demographic proxy** (`compute_sparc_demographic_proxy`, diagnostic only):
- log Σ = log₁₀(SBdisk) — stellar surface density
- f* = M* / (M* + 1.33 × M_HI) — stellar mass fraction (L36-based)
- R_HI/R_d — HI-to-optical disk scale-length ratio
- D = (z_logΣ + z_f* − z_RHI/Rd) / 3 (robust z-scores, minmax normalized to [0,1])

**SPARC M74 coherence score** (`compute_sparc_m74_coherence`, paper-facing):
- Five-component average: (1−f_gas), f_star, normalized log₁₀ Σᵦ, 1/(1+R_HI/Rdisk), taxonomy prior
- Plus radial-curve bonuses (±0.15 flatness, ±0.10 curve_flatness) for 25 resolved galaxies
- Quartiles via `pd.qcut(..., duplicates='drop')` giving n=22/22/21/22
- This M74 score used historical `coherence_score` terminology; the paper treats it as an organization proxy.

**xGASS organization proxy** (`compute_xgass_coherence_proxy`):
- C = 0.35 × (incl/90°) + 0.35 × minmax(W50cor) + 0.30 × (1 − f_atm)
- `f_atm` is inherited from the M75 input column `f_atm_total`, not recomputed from the visible mass columns.

**Ablation variants** (7 total, 2 alias pairs removed):
| Variant | Definition |
|---------|------------|
| `full` | 0.35×incl + 0.35×w50 + 0.30×(1−f_atm) |
| `incl_fatm_no_w50` | 0.5×incl + 0.5×(1−f_atm) (= alias `no_raw_w50`, removed) |
| `w50_fatm_no_incl` | 0.5×w50 + 0.5×(1−f_atm) (= alias `no_incl`, removed) |
| `incl_only` | incl/90° |
| `w50_only` | minmax(W50cor) |
| `gas_only` | 1 − f_atm |
| `no_gas` | 0.5×incl + 0.5×w50 |

### The 363/320 galaxy split

Of the 363 xGASS galaxies with clean HI detections, **320** have derived Vrot inputs
(columns `V_rot_HI_am`, `incl`, `f_atm`). The remaining 43 galaxies lack these
measurements and are excluded from proxy analysis (which requires `incl`, `W50cor`,
`f_atm`). The 363-entry table includes all quality-1 HI detections; the ablation
and controls analyses run on the 320-galaxy subset (or 363 for inclination-only
variants).

## Reference outputs

- `arxiv_revised_analysis.json` — Full reference analysis results. The `paper_values`
  block declares the canonical manuscript numbers: xGASS full ablation Q4/Q1=0.41065,
  SPARC M74 Q4/Q1=0.65064, ΔR²=0.00348. Contains two sets of controls ΔR² values;
  the paper uses `null_models.combined_controls` (ΔR²=0.00348, controls: lg_mbar,
  incl, hi_flag). The `quality_controls` entry (ΔR²=0.00213) uses a different control
  set (lg_mbar, incl, b/a, HI_conf_flag). Bootstrap confidence intervals in this
  JSON were generated with seed 42; the pipeline uses RANDOM_SEED=202606, so pipeline
  bootstrap CIs differ slightly.

- `sparc_m74_btfr_summary.json` — Paper-facing SPARC M74 summary. The SPARC M74
  proxy table (`sparc_m74_proxy_table.csv`) reproduces Q4/Q1 = 0.6506373440 exactly.
  The current diagnostic proxy table (`sparc_proxy_table.csv`, columns
  `sparc_current_org_proxy` / `sparc_current_org_proxy_quartile`) uses a different
  proxy definition and will not reproduce the M74 ratio.

- `sparc_m74_quartile_scatter.csv` — Exact per-quartile scatter values from M74.
