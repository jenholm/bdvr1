# Data directory

## Structure

```
data/
├── raw/         Source input files (SPARC quality1 inputs, xGASS coherence scores)
├── external/    Manually downloaded external catalogs (not committed)
├── processed/   Cleaned intermediate tables (generated by scripts)
└── derived/     Paper-specific derived products (generated by scripts)
```

## Committed input files

This repository includes the following input files for reproducibility:

- `data/raw/sparc/sparc_quality1_rich_inputs.csv` — SPARC quality-1 galaxies (87 galaxies, Q=1). Source: Lelli+2016.
- `data/raw/xgass/m75_xgass_coherence_scores.csv` — xGASS H I-detected galaxies with coherence scores and BTFR residuals (363 galaxies). Derived from the xGASS catalog.

## Obtaining original catalogs

### SPARC

Download the full SPARC catalog from http://astroweb.cwru.edu/SPARC/

### xGASS

Download the xGASS representative sample from https://xgass.astro.umd.edu/

Place additional files in `data/raw/xgass/` and run:

```bash
python scripts/01_prepare_inputs.py
```

## Generated files

Run `make prepare` to generate:
- `data/processed/sparc_clean.csv` — Cleaned SPARC analysis table
- `data/processed/xgass_clean.csv` — Cleaned xGASS analysis table

Run `make analysis` to generate derived tables.

## Derived tables (committed)

The following derived tables are committed and redistributable:

- `data/derived/sparc_proxy_table.csv` — SPARC proxy scores (87 galaxies)
- `data/derived/xgass_proxy_table.csv` — xGASS proxy scores (363 galaxies)
- `data/derived/btfr_residuals_sparc.csv` — SPARC BTFR residuals
- `data/derived/btfr_residuals_xgass.csv` — xGASS BTFR residuals
- `data/derived/proxy_ablation_results.csv` — 9-variant ablation results
- `data/derived/sparc_btfr_summary.json` — SPARC BTFR scatter summary from main analysis
- `data/derived/arxiv_revised_analysis.json` — Reference results from arxiv revised analysis

## Reference outputs

- `data/derived/arxiv_revised_analysis.json` — Full reference analysis results for reproducibility testing
- `data/derived/sparc_btfr_summary.json` — SPARC scatter summary (M74 BTFR coherence audit)
