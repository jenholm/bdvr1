# M29 four-stage taxonomy rebuild

M29 reformulates the UGC07125 problem. UGC07125 is no longer treated as the juvenile prototype. It is classified as a failed/stalled boundary-object candidate unless future resolved data overturn the current scalar diagnostics.

This run does not rescue M20-M28. It is not a submission-grade claim by itself. It does not create a submission-grade claim by itself. It is a taxonomy/triage layer that says which physical question to ask next.

## Four-stage boundary taxonomy

1. `juvenile_forming`: gas-rich/diffuse/low-stellar systems that are not catastrophically under-rotating.
2. `failed_stalled`: gas-rich/diffuse/extended systems that are severely under-rotating for their catalogued baryonic mass; candidate nonparticipating-gas or stalled-disk systems.
3. `disturbed_or_measurement_contaminated`: severe mismatches that must be quarantined behind geometry, uncertainty, warp, distance, or curve-quality checks.
4. `true_falsifier_candidate`: clean severe mismatches without failed/stalled indicators. These require resolved H I and geometry checks before any true falsifier claim.

Rows outside the boundary problem are labeled `mature_or_control`.
