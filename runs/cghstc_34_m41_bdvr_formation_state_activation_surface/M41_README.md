# M41 BDVR Formation-State Activation Surface

M41 implements an activation-surface diagnostic for baryon-coupled reservoir response.
BDVR activation is not triggered by raw baryonic mass alone; the proposal is that
activation follows organized baryonic structure, rotational coherence, gas conversion,
and low reservoir-disorder suppression.

This is not a velocity-curve fit and does not claim BDVR is real. It emits concrete
missing-data predictions for gas-rich, disturbed, failed, juvenile, and mature rows.
