# M42 Optimal Transition Boundary Search

M42 searches candidate BDVR transition thresholds. It is not another generic
classifier and it does not replace the M36 mature velocity endpoint. M41 is used
only as scaffolding/input features.

The branch asks where current data place the transition from gas-rich or
unsettled systems to organized, long-lived, BDVR-active disks. Missing SFE,
depletion-time, and H I-asymmetry labels are emitted as null/unconstrained
thresholds plus falsifiable missing-observable predictions.
