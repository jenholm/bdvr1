CGHSTC-34 M6i: Dynamic Reservoir Governor Toy Diagnostic
====================================================================

Status: complete.

Purpose
-------
Test the equilibrium shadow of a dynamic reservoir governor.

The motivating idea is that reservoir coupling may rise as gas becomes organized into stellar/baryonic structure and bleed down when gas remains diffuse or HI structure is extended/disturbed.

This stage is diagnostic only.
No model term is accepted.
No parameters are retuned.
No outliers are deleted.

Dynamic toy equation
--------------------
dA/dt = alpha * activation * (1 - A) - (beta * gas + gamma * disorder) * A

Equilibrium form
----------------
A* = alpha * activation / (alpha * activation + beta * gas + gamma * disorder)

Interpretation
--------------
activation represents stellar organization.
gas represents diffuse gas burden.
disorder represents HI extent, low surface density, or a diagnostic symptom.

Important boundary
------------------
The low-V-for-mass disorder term is not an independent physical driver.
It is included only as a ceiling diagnostic because M6g showed that low-V-for-mass dominates the static governor correlations.
