# M43 Baryonic Maturity Metric and End-State Inversion

M43 builds a baryon-only maturity coordinate. The branch reverse-engineers the
visible baryonic platform a juvenile or failed system would need to reach to
become a long-lived mature disk endpoint.

Dark halo parameters, BDVR distribution, reservoir-field geometry, missing-mass
profiles, velocity residuals, and inferred activation are not used to define the
maturity metric. Velocity residuals are outcomes for later testing only.

The current scalar artifacts lack decisive resolved Vrot/sigma, approaching vs
receding mismatch, H I asymmetry, SFE, depletion time, and Fourier/centroid/PA
shape observables. M43 reports those as required boundary observables rather
than fabricating them.
